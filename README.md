## 🇧🇷 Português

### O que é este repositório?

Este repositório documenta minha jornada de aprendizado partindo do **absoluto zero** — sem conhecimento prévio de programação, lógica, Python ou SQL — com o objetivo de me tornar um **Analista de Dados**.

Uso o **Claude Code** como parceiro de aprendizado: cada notebook foi construído com o conceito explicado antes do código, progressão lógica sem pular etapas e exemplos do cotidiano. O código é em inglês, as explicações em português.

### Por que tornar isso público?

Porque aprendi que ver a jornada de outra pessoa — com erros, dúvidas e progresso real — é mais valioso do que tutoriais polidos. Se você também está começando do zero, este repositório pode ser uma referência honesta.

## 🇺🇸 English

### What is this repository?

This repository documents my learning journey starting from **absolute zero** — no prior knowledge of programming, logic, Python, or SQL — with the goal of becoming a **Data Analyst**.

I use **Claude Code** as a learning partner: each notebook is built with the concept explained before the code, logical progression with no skipped steps, and real-world examples. Code is in English, explanations in Portuguese (pt-BR).

### Why make it public?

Because seeing someone else's journey — with real mistakes, questions, and progress — is more valuable than polished tutorials. If you're also starting from scratch, this repository can be an honest reference.

### Roadmap

**Phase 1** — Programming Fundamentals (Python)  
**Phase 2** — Databases (SQL, MySQL, PostgreSQL, MongoDB)  
**Phase 3** — Data Analysis (pandas, numpy, statistics)  
**Phase 4** — Visualization (matplotlib, seaborn, plotly, Power BI)  
**Phase 5** — Web & Deploy (HTML/CSS, Django)  
**Phase 6** — Next level (Machine Learning, Streamlit)  

### How to use

```bash
git clone https://github.com/pspellegrini/data-analysis-road.git
cd data-analysis-road
pip install jupyter
jupyter notebook
```
